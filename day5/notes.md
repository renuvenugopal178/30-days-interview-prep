# Day 5 Notes

## Dictionary
Stores key-value pairs. Lookup is usually O(1).

## Set
Stores unique values. Membership checking is usually O(1).

## Frequency Map
A dictionary used to count occurrences.

## Two Pointers
Two indexes move through an array or string, often from both ends.

## Sliding Window
A moving range in an array or string. It is useful for substring and subarray problems.

## Patterns
- Duplicate detection: set
- Pair lookup: dictionary
- Palindrome: two pointers
- Longest unique substring: sliding window + dictionary
