# Day 6 Notes — SQL and DBMS

## Database
An organized collection of data.

## DBMS
Software used to create, store, retrieve, and manage databases.

## Keys
- Primary key: uniquely identifies a row.
- Foreign key: creates a relationship between tables.

## CRUD
- Create: INSERT
- Read: SELECT
- Update: UPDATE
- Delete: DELETE

## SQL Clauses
- WHERE filters rows.
- ORDER BY sorts results.
- GROUP BY creates groups.
- HAVING filters groups.
- JOIN combines related tables.

## ACID Properties
- Atomicity: a transaction completes fully or not at all.
- Consistency: data remains valid according to rules.
- Isolation: transactions do not interfere incorrectly.
- Durability: committed data remains saved.

## Normalization
Organizing tables to reduce duplicate data and update errors.
